Package glossary¶

Dictionary of any terms and their interpretations app package.

The application is available only to a user with administrator rights. The application is only accessible through the Django admin interface. It has the functionality of adding terms and displaying a list of them. Currently, development of the package is suspended. In the future, work on the project is expected to be resumed.

Package task¶

Self-education exercise app package.

There are two approaches to performing exercises:

  • self-education without incentives

  • education using stimulation

Education using stimulation¶

Points are awarded for correct execution of the exercise. A limit has been set on the number of points that can be received per day. From day to day, the points received accumulate.

Points can be deducted, for example, when a parent rewards a child for studying. This way, the child becomes motivated to do the exercises.
