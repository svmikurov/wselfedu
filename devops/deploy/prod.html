Production deploy¶

Clone repo:

bash:¶
git clone git@github.com:svmikurov/wselfedu.git

Go to project:

bash:¶
cd wselfedu/

Add .env and .env.postgres:

bash:¶
cp .env.example .env && \
cp .env.postgres.example .env.postgres

In the .env and .env.postgres files fill the empty values of example below or leave the default values. Value POSTGRES_DB is POSTGRES_NAME value.

bash:¶
nano .env
.env¶
SECRET_KEY=
DEBUG=0
ENVIRONMENT=production

# Database settings
POSTGRES_NAME=
POSTGRES_USER=
POSTGRES_PASS=
POSTGRES_HOST=wse-db-postgres
POSTGRES_PORT=5432
bash:¶
nano .env.postgres
.env.postgres¶
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=

Build and Up Docker:

bash:¶
 make build up

Make migrations:

bash:¶
make migrate

Make collectstatic:

bash:¶
make collectstatic

This is all.
